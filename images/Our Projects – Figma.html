You can now export code from code blocks as separate local files.

Compatible with Figma for VS Code extension.


How to use:

  1. Select code block(s);
  2. Run the plugin;
  3. Save the file to disk.


Supported objects:

  • Сode blocks.


Note:

Code block is an object from FigJam (read more at Figma Learn and Figma Developers).


Tip:

The name of the exported file will match the first line of the code.
